Калькулятор металла позволяет рассчитать приблизительную массу по стандартным размерам труб круглых и профильных, уголка, швеллера, балки, листа произвольных размеров.
Для выполнения расчета необходимо:
1. Выбрать необходимый профиль металлопроката.
2. В окне расчетов ввести указанные стандарные размеры металлопроката, выбранного типа.
3. Нажать кнопку Calculate. Под надписью Weight появится значение расчета в кг.
